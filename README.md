# Secure Login System

## Description
This project is a secure login system developed using Flask and bcrypt.

## Features
- User Registration
- Secure Login
- Password Hashing using bcrypt
- Session Management
- Logout Feature
- Basic Input Validation

## Technologies Used
- Python
- Flask
- bcrypt

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Project Structure

- app.py
- requirements.txt
- templates/login.html
- templates/register.html
